# AI Dependency Risk Analyzer
AI-powered scanner detecting vulnerabilities, license issues, and supply-chain attacks.

## Tech Stack
- MERN, Python, Docker, Kubernetes, Jenkins, AWS

## Key Features
- AI-powered scanner detecting vulnerabilities, license issues, and supply-chain attacks
- LLM-based anomaly detection with report automation and heatmap dashboard
- Cloud-native microservices improving threat detection accuracy by **35%**

## Installation
1. Clone the repo: `git clone <repo-link>`  
2. Install dependencies: `npm install`  
3. Start backend and AI services  
4. Start frontend dashboard  

## Usage
- Upload a project dependency file to detect security risks  
- View risk reports and heatmap analytics on the dashboard
